Mon rôle

Mes missions en alternance

Au sein de la DSI, dans une mission data & BI avec des outils industriels.

📊
Power BI — Reporting
Création de tableaux de bord de réconciliation (voitures balai) comparant les données logistiques et RH entre plusieurs systèmes sources. Rédaction de mesures DAX et de requêtes Power Query M.
Power BI DAX Power Query M Reporting
🔄
Migration Cognos → Power BI
Recette de migration du module réception logistique de Cognos vers Power BI. Rédaction du cahier de recette, validation des données et analyse des anomalies de températures.
Recette Cognos Validation données Word
🗄️
SQL — Analyse Snowflake
Analyse et exploration de données métier via SQL dans l'environnement Snowflake, avec DBeaver. Requêtes SELECT, JOIN, GROUP BY, agrégations et particularités Snowflake.
SQL Snowflake DBeaver Analyse
👥
Rapports RH — Workday
Tableau de bord de comparaison des données RH entre Workday et Acciline (SST), avec enrichissement géographique via un référentiel des départements français.
Workday Acciline SharePoint RH Data
📝
Documentation technique
Rédaction de documentations Word et PowerPoint pour les outils développés, mise à jour des cahiers de recette, rapports de validation et communication avec les équipes.
Word DOCX PowerPoint Documentation
🌡️
Anomalies & validation
Analyse des écarts de températures dans les rapports Power BI vs Cognos. Identification de la cause (agrégation Sum), reformulation du rapport de validation pour les équipes métier.
Validation Agrégation Anomalies Rapport
